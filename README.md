# Discord Helper Bot

A comprehensive Discord helper bot with extensive features including ticket systems, giveaways, credit economy, moderation tools, and much more.

## Features

- **Ticket System** - Create and manage support tickets
- **Giveaways** - Host automated giveaways with various settings
- **Credit System** - Economy system with credits, transactions, and rewards
- **Mod System** - Complete moderation tools (bans, mutes, joins, logs)
- **Suggestions** - Community suggestion system
- **AutoRole** - Automatic role assignment
- **Welcome Setup** - Customizable welcome messages
- **Wipe Setup** - Data management tools
- **Sticky Messages** - Pin important messages
- **Sticky Embed Message Editor** - Create and edit embed messages
- **Boost Rewards** - Reward server boosters
- **TagHype Squad** - Special tagging features
- **Events** - Interactive events (dino, number, vault, chat)
- **Mystery Box** - Loot box system with rewards
- **Mystery Box Reward Claiming** - Claim mystery box rewards
- **Polls** - Create and manage polls
- **Embed Builder** - Build custom embeds
- **Dropdown Builders** - Create dropdown menus
- **News Poster** - Post news and announcements
- **Admin Pay Tracker** - Track admin payments
- **SellAuth Integration** - Integration with SellAuth platform
- **Global Coupons** - Create and manage coupons
- **Credit Management** - Add/redeem credits for users
- **Invoice Tracking** - Track invoices and payments
- **Automatic Dono Confirmations** - Automatic donation confirmations

## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and configure your settings
4. Run the bot:
   ```bash
   npm start
   ```

## Configuration

Edit the `.env` file with your bot token and other settings:

- `BOT_TOKEN` - Your Discord bot token (required)
- `OWNER_ID` - Your Discord user ID
- `PREFIX` - Command prefix (default: !)
- And many more settings...

## Commands

The bot includes a comprehensive command system. Use `!help` to see all available commands.

## Database

This bot uses SQLite for data storage. The database will be automatically created on first run.
